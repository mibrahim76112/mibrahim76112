# Hi, I'm Ibrahim 👋

## About

I am a Master's student in Electrical and Computer Engineering at the University of Manitoba, Canada.

My work focuses on building intelligent connected systems that combine embedded devices, wireless communication, and machine learning. I am interested in systems where real-world data is collected from hardware, transmitted through communication networks, processed through software, and used to make better decisions.

My interests include:
- ⚙️ Embedded systems and connected devices
- 📡 Wireless communication and network optimization
- 🤖 Machine learning for perception, control, and decision-making


## Embedded Systems

- ESP32 and ESP32-S3 firmware with ESP-IDF and FreeRTOS
- Wi-Fi telemetry, reconnection logic, device health monitoring, and command handling
- Raspberry Pi gateway services, REST/gRPC APIs, WebSocket dashboards, and OTA-ready architecture
- Sensor integration, actuator workflows, hardware interfaces, debugging, testing, and system documentation

## Wireless Systems

- 5G and 6G wireless system modeling and simulation
- Beamforming, resource allocation, channel modeling, and performance evaluation
- Intelligent surfaces, multi-user systems, and deep unfolding for wireless optimization
- Simulation workflows using Python, MATLAB, PyTorch, and technical reporting

## Machine Learning

- Reinforcement learning for control, decision-making, and optimization
- Multi-agent learning for UAV and networked systems
- Computer vision for object detection, tracking, and embedded perception
- Jetson Nano deployment, TensorRT optimization, model evaluation, and baseline comparison
